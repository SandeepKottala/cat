Welome to Blockchain Technology

A blockchain is a growing list of records, called blocks, that are securely linked together using cryptography.
Each block contains a cryptographic hash of the previous block, a timestamp, and transaction data (generally represented as a Merkle tree, where data nodes are represented by leafs).

Types of Blockchains:

  • Public Blockchains
  • Private Blockchains
  • Hybrid Blockchains
  • Sidechain
